# webt2310: HTML & CSS review

This folder contains a minimal static-site exercise used for HTML/CSS practice.

How to preview locally:

```bash
cd htmlandcssreview
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Edit `index.html` and files under `css/` and `images/` to update the site.
